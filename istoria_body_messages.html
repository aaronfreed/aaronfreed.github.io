Marathon Istoria: Conversations with the Dead

This is a list of all possible messages that can obtained by communicating with the Yellow Crystal in the Aleph One scenario Marathon Istoria. A document of this type will inevitably contain massive gameplay and story spoilers, and consulting a guide of this sort can easily be considered ‘cheating’; turn back now if this bothers you.

A few notes:

  • If a body doesn’t respond to a topic, none of the bodies in the game will respond to it.
  • Multiple keywords that produce the same response are grouped together.
  • The ending assigns points for each unique response from a body.
    • If you die without having saved since a conversation, you’ll need to repeat it to get the points.
  • Numbering is per the game’s internal code. The URLs that point to each body use the game’s internal reference indicator for that body following the # sign.

Please contact me if you notice any errors or omissions.
